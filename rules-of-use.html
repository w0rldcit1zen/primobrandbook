07 · RULES OF USE

The fundamentals stay at the heart.

Systems scale the story; they never replace it. These rules exist so that everything Primo makes — at any speed, in any format, by any tool — still sounds like Primo and can still be received by everyone.

Received by everyone — measured, not asserted

A story only counts if it is received. So every claim below is a measured number, re-provable at any time by running verify.py against tokens.json (both live in the primo-brand skill). Conformance: WCAG 2.2 AA — also the standard our institutional deliverables are procured against (EN 301 549, Chapter 10).

Do not

  • Do not put bright Signal Green on a light background — 2.54:1 is a signal nobody can read. Use Signal Green Deep.
  • Do not reintroduce Teal #4A7C7D (3.60:1) or Amber #D4A373 as an accent (ΔE 3.0 under protanopia).
  • Do not exceed five chart series.
  • Do not use 13px for body copy.
  • Do not claim the golden ratio is objectively beautiful.
  • Do not hardcode a hex value anywhere. It lives in tokens.json.

Governance — how the book stays true

The source of truth is the primo-brand skill (tokens.json + BRAND.md + verify.py). This website is generated from those tokens. The sync rule: any token change → run build.py → regenerate these pages → update the Canva kit and primo-motion → re-run verify.py. If verify fails, the brand has drifted — stop and fix before publishing anything.

Voice — the short canon

Conviction, then numbers. Thesis first; beliefs declared plainly; hard figures over adjectives; em-dashes and semicolons for momentum; specificity over superlatives — name the flags on the beach, the church bells, the phone calls between cousins. Serious, never breathless. Never "It's not just X — it's Y."
